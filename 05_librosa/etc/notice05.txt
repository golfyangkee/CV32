1. PyDub  : 다양한 오디오 형식을 지원하며, 오디오 파일의 편집이나 변환에 유용
https://github.com/jiaaro/pydub

2. Music21 :음악 이론, 음악 분석, 그리고 악보 생성에 사용 - 생성형 AI 때문에 뜨고 있음
http://web.mit.edu/music21/

3. FluidSynth : MIDI 이벤트를 실시간 오디오로 렌더링 - 음악 파일을 가져다가 여러가지 다양한 섹션으로 수정도 하고 변환도 하고 서비스도 하는 사이트 요즘 뜨고 있음
https://www.fluidsynth.org/

4. pretty_midi : MIDI 파일을 읽고 쓰고, 분석
https://github.com/craffel/pretty-midi

5. Essentia : 매우 광범위한 오디오 분석 기능을 제공하며, 고성능이 필요한 상황에서도 적합 - 악기를 다루는 회사들이 해당 사이트 라이브러리 가져다가 만듦 뜨고 있음
https://essentia.upf.edu/

6. Madmom :음악 정보 검색과 오디오 신호 처리에 특화된 라이브러리. 특히 리듬 분석에 강점
https://github.com/CPJKU/madmom

7. scipy.io.wavfile : SciPy 라이브러리 내에 있는 wav 파일 입출력 모듈, 기본적인 오디오 처리에 사용 - 수업할 때 쓸거
https://docs.scipy.org/doc/scipy/reference/io.html

음악의 장르를 자동분류  : MFCC 특성을 사용한 머신러닝 분류 작업
ex) 데이터 수집        -> 데이터 전처리  ->    [모델 훈련  -> 모델 평가(성능평가) ]    -> 모델적용
 장르별 음악파일 수집  ->  MFCC 특성추출  ->   [ (머신러닝  / 딥러닝) =  분류기  ]  ->  실제새로운음악파일을 주면 장르 분류
================================< 기본 전처리 >==========================================================================
1. 기초 예제
Waveform 그리기: 오디오 파일에서 waveform을 그리는 기본적인 예제
Spectrogram 생성: FFT(Fast Fourier Transform)나 STFT(Short-Time Fourier Transform)를 사용하여 spectrogram을 그리기

2. 특성(특징) 추출 예제
MFCC 계산: Mel-frequency cepstral coefficients를 계산하고 그래프로 표시
Chroma features 계산: Chroma 특성을 추출하여 그래프로 표시

3. 음악 정보 검색 (Music Information Retrieval, MIR)
템포 추정: 오디오에서 템포(beat)를 추정
음고 분석: 기본 주파수(F0)나 피치를 추출

==========================< 알고리즘 적용 > =============
4. 분류 문제 예제
장르 분류: 여러 음악 장르에 대해 MFCC나 다른 특성을 사용하여 분류 모델 생성
음성/음악 구분: 음성과 음악을 구분하는 분류 모델 생성 ->  성별 구분 / 나이구분

5. 고급 분석 예제
음성 인식: 오디오에서 특정 단어나 구절을 인식
감정 분석: 음성에서 감정을 분석 (ex: 행복, 슬픔, 분노 등)

6. 시계열 데이터 분석
Onset Detection: 음악이나 음성에서 onset(시작 부분)을 감지
Beat Tracking: 비트를 추적하고 그 위치를 표시

오디오의 특징 추출:

스펙트럼 (Spectrum): 주파수 도메인에서 오디오 신호를 분석

MFCC (Mel-Frequency Cepstral Coefficients): 음성 및 오디오 분석에서 주로 사용되는 특징(1)

STFT (Short-Time Fourier Transform): 시간에 따른 주파수 변화를 나타내며, 스펙트로그램을 생성하는 데 사용(2)

Chroma Features: 음악 분석에서 주로 사용되며, 비교할  다른 피치 클래스에 대한 에너지 분포를 나타냄(3)

(1) (2) (3) 딥러닝할 때 필수

Zero Crossing Rate: 신호가 얼마나 빨리 진동하는지 표현

RMS Energy: 오디오 프레임의 에너지를 계산

Pitch & Cadence: 음성의 높낮이와 리듬, 또는 간격을 분석

Spectral Contrast: 주파수 범위 내에서 뾰족한 봉우리와 평탄화  차이를 계산
================================================================================
비트  :  0, 1 단위   -> 정보 양 , 스토리지 용량 체크
    < Bit Depth : 단일샘플에 할당된 비트수  >
        - 비트수를 기준으로 잡음체크  (디지털오디오)
        - 특정픽셀의 색상표현 _품질개선   ( 디지털이미지 )

주파수  :  파동이나 진동이 일정한 시간에 몇번이나 발생하는 표현하는 물리량,-> Hz   => 음의 높낮이를 결정
펄스(Pulse) = 폭, 높이, 주기, 펄스간격:일시적인 진폭의 변화나 신호의 짧은 변동 ( 디지털 신호처리, 통신 , 의료장비(펄스 산소계)  ,,,)
================================================================================================
- 파동=Wave : 에너지나 정보를 전달하는 형식, 물리적 전달 체( 공기, 물 ), 주기적인 진동 또는 변동을 가진다.
     진폭= 파동의 크기, 주파수 , 파장= 연속적인 파동이 반복되는 사의의 거리 , 속도 = 파장의 체크
- 진동 : 기준에 의해서 물체가 움직이는 현상 (진폭, 주파수, 위상 : 진동의 상태를  각도의 단위  )

이미지의 특징 추출:
SIFT (Scale-Invariant Feature Transform): 스케일과 회전에 불변하는 특징 포인트를 추출

SURF (Speeded-Up Robust Features): SIFT보다 계산 효율이 좋은 특징 추출

HOG (Histogram of Oriented Gradients): 이미지의 지역적 그래디언트 방향을 히스토그램으로 표현
                            주로 사람이나 동물을 탐지하는 데 사용

ORB (Oriented FAST and Rotated BRIEF): 계산 효율이 좋으면서 회전과 스케일에 불변한 특징을 추출

Color Histograms: 이미지 내의 색상 분포

Texture Features: Gray Level Co-occurrence Matrix (GLCM), Local Binary Patterns (LBP) 등을 통해 텍스처 정보를 추출

Convolutional Layers: 딥러닝 모델, 특히 CNN(Convolutional Neural Networks)에서 자동으로 이미지 특징을 추출

Edge Detection: Canny, Sobel, Laplacian 등의 방법을 사용해 이미지에서 가장자리 정보를 추출

스펙트럼 :  시간과 주파수 도메인에서 오디오 신호의 특성을 표현한다.
D = librosa.amplitude_to_db(np.abs(librosa.stft(y)), ref=np.max)
librosa.stft(y) :  원본 데이터를 시간에 따라 푸리에 연산을 통해 (시간-주파수)를 리턴한다. 단,복소수를 포함할 수 있다.
np.abs()  : 절대값 변환
amplitude_to_db() : 진폭의 절대값을 디시벨(db)로 변환한다.
ref=np.max   :   스펙트럼의 최대값을  0db로 하겠다.
D   :  db단위로 표현된 스펙트럼이다.
==============================================================
푸리에 연산(Fourier Transform)  : 기저함수는 일반적으로 복소수. 지수 함수의 형태를 가진다.
     - 복잡한 신호를 단순한 주기적인 함수 (사인, 코사인)의 합으로 분해하는 것이다.
     - 기저함수 : 주어진 함수 공간 내에서 임의의 함수를 표현할 수있는 최소한의 함수집합
     - 기저함수를 통해서 복소수 형태로 주어진 결과를 신호의 주파수 성분과 관련이 있다.

===================================================================
FT  :연속시그널   -> 연속 주파수 스펙트럼   : 시그널 분석할 때 수학에서는 적분을 사용해서 변환을 계산하다.
DFT : 이산 시그널  ->  이산 주파수 스펙트럼  : 샘플링된 시간 도메인을 변환할때 사용한다.   / 수학에서 합계로 계산한다.
Fast FT : 딥러닝 알고리즘사용시 대중화   : DFT를 시그널로 조각을 내서 재귀적으로 연산하는 방법
